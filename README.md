# MITRA CUAN POS

Web-based Point of Sales (POS) application with local storage persistence.
Created by **RMD TECH**.

## Features
- **Dashboard**: Overview of sales, orders, and customers.
- **POS**: Product selection, cart management, and checkout.
- **Transactions**: History of all transactions.
- **Customers**: Customer database with spending history.
- **Settings**: Application configuration.
- **Local Storage**: Data persists across page reloads.

## Tech Stack
- React (Vite)
- Tailwind CSS
- Lucide React (Icons)
- LocalStorage (Database)

## How to Run

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the development server:
    ```bash
    npm run dev
    ```

3.  Open your browser at `http://localhost:5173`.

## Build for Production

```bash
npm run build
```
